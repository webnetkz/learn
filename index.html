MySQL

MySQL свободная реляционная система управления базами данных

Гибкость СУБД MySQL обеспечивается поддержкой большого количества типов таблиц: пользователи могут выбрать как таблици типа MyISAM, поддерживающие полнотекстовый поиск, так и таблици InnoDB, поддерживающие транзакции на уровне отдельных записей. Более того, СУБД MySQL поставляется со специальным типом таблиц EXAMPLE, демонстрирующим принципы создания новых типов таблиц. Благодаря открытой архитектуре и GPL-лицензированию, в СУБД MySQL постоянно появляются новые типы таблиц.

26 февраля 2008 года Sun Microsystems приобрела MySQL AB за 1 млрд. $, 27 января 2010 года Oracle приобрела Sun Microsystems за 7.4 млрд. $ и включила MySQL в свою линейку СУБД.

Происхождение MySQL

MySQL возникла как попытка применить mSQL к собственным разработкам компании: таблицам, для которых использовались ISAM — подпрограммы назкого уровня. В результате был выработан навый SQL-интерфейс, но API-интерфейс остался в наследство от mSQL. Логотип MySQL в виде дельфина носит имя Sakilla.

Лицензирование

MySQL имеет двойное лицензирование. MySQL может распространяться в соответствии с условиями лицензаа GPL. Однако по условиям GPL, если какая-либо программа использует библиотеки (или включает в себя другой GPL-код) MySQL, то она тоже толжна распространяться по лицензии GPL. Это может расходиться с планами разработчиков, не желающих открывать исходные тексты своих программ. Для таких случаев предусмотренна коммерческая лицензия, которая также обеспечивает качественную сервисную поддержку. Для свободного программного обеспечения Oracle предоставляет отдельное исключение из правил, явным образом разрешающее использование и распространение MySQL вмете с ПО, распространяемым под лицензией из определенного Oracle списка.
